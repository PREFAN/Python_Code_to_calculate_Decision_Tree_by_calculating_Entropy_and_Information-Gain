# Python_Code_to_demonstrate_Decision_Tree_by_calculating_Entropy_and_Information-Gain
This Python script demonstrates the core concepts of Decision Tree learning by manually calculating Entropy and Information Gain using the classic "Play Tennis" dataset. The project serves as an educational implementation of the ID3 (Iterative Dichotomiser 3) algorithm’s fundamental calculations from scratch, without relying on any machine learning library.
The code begins by creating a small, well-known Play Tennis dataset using pandas. This dataset contains 14 instances with four weather-related features — Outlook, Temperature, Humidity, and Wind — and a binary target variable 'Play' indicating whether tennis was played or not on that day.
The script defines a custom entropy() function that calculates the Shannon entropy of any given subset based on the target variable. Using the mathematical formula for entropy, it measures the impurity or uncertainty in the data. The overall entropy of the entire dataset (E(S)) is calculated first, providing a baseline measure of disorder before any splitting.
The main focus of the code is on the Outlook feature. It groups the dataset by each unique value of Outlook (Sunny, Overcast, and Rain), calculates the entropy of the 'Play' target within each group, and then computes the weighted average entropy based on the proportion of samples in each subgroup. This weighted entropy represents the expected entropy after splitting the data on the Outlook attribute.
Finally, the script calculates the Information Gain for the Outlook feature by subtracting the expected entropy (after the split) from the original overall entropy. Information Gain quantifies how much uncertainty is reduced by splitting the dataset on a particular feature. In this case, Outlook is analyzed to determine how effectively it can separate the "Play" and "No Play" classes.
Through detailed print statements, the code shows every step of the calculation — including individual group entropies, their weights, the expected entropy after splitting, and the final Information Gain value. This transparency makes the script highly educational, allowing users to clearly understand how Decision Trees decide which feature to split on at each node.
Overall, this project provides a clear, step-by-step walkthrough of the mathematical foundations behind Decision Tree algorithms. It reinforces key concepts such as entropy as a measure of impurity, weighted entropy, and information gain as a criterion for feature selection. This implementation is particularly valuable for students and beginners who want to understand what happens "under the hood" of popular machine learning algorithms like ID3 and C4.5 before using high-level libraries such as scikit-learn.
